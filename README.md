# Blinkit Delivery Dashboard 📦

This Power BI dashboard provides an analysis of Blinkit's delivery performance, sales trends, and customer metrics.

## 🔍 Key Features:
- Daily order trends & revenue
- Average delivery time by city
- Most ordered product categories
- Filters by city, month, and delivery status
- KPIs: Orders, Revenue, Avg. Delivery Time, Cancellations

## 🛠 Tools Used:
- Power BI
- Excel / CSV
- DAX for custom measures
